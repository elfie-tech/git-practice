## Terminal commands
- `ls`    : list files
- `ls -a` : list files including hidden ones
- `cd` : change directory
- `cd ~` : go to home directory
- `mkdir` : make directory
- `touch` : adds files
- `pwd` : present working directory (where you're at)
- `cd ..` : goes to previous directory
